# Click-to-Ad Public Status

## Current Scope

The public version currently covers the core demo flow:

- Analyze a product URL
- Extract product information and images
- Edit the product kit before generation
- Choose a style template
- Preview the final prompt
- Submit a video generation task
- Poll task status and display the result

## Implemented Capabilities

- Amazon, Shopify, and generic webpage parsing
- Structured product extraction with fallback behavior
- Local image upload for supplementing product visuals
- Prompt construction from reusable ad templates
- Async video task submission and polling
- Mock video fallback when video API access is unavailable

## Public Repo Notes

- This file is a public status summary, not an internal changelog
- Private deployment details and iteration history are intentionally excluded
