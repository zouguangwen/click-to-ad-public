# Click-to-Ad: AI 驱动的一键电商视频广告生成平台

> 从产品链接到成品视频广告，全程 AI 自动化，3 分钟内完成。

---

## 一、项目背景与市场洞察

### 痛点

跨境电商卖家（Amazon/Shopify）日常面临大量短视频广告素材需求（TikTok、Instagram Reels、YouTube Shorts），但传统制作流程存在明显瓶颈：

| 传统流程 | 耗时 | 成本 |
|----------|------|------|
| 拍摄 + 剪辑 | 1-3 天 | $200-2000/条 |
| 模板工具（Canva 等） | 2-4 小时 | $30/月起 |
| 外包设计师 | 3-7 天 | $100-500/条 |

**核心矛盾**：卖家需要高频、低成本、多风格的视频素材，但现有方案无法同时满足效率和质量。

### 机会

视频生成模型（Sora、Kling、Vidu）的进展，使 **"AI 原生的视频广告工厂"** 成为可能。不同于通用视频生成工具，本产品聚焦 **电商广告垂直场景**，通过 **结构化 prompt 模板 + 产品数据自动提取** 降低使用门槛。

### 目标用户

- **主要用户**：跨境电商运营 / 独立站卖家（海外市场）
- **次要用户**：DTC 品牌营销团队、MCN 机构素材批量生产

## 二、产品定义

### 一句话描述

**粘贴商品链接 → AI 自动提取素材 → 选择广告风格 → 生成成品视频**

### 核心价值主张

1. **零设计门槛**：无需上传素材包，粘贴 URL 即自动提取产品图、品牌色、文案
2. **专业广告模板**：8 套经过 prompt 调优的视频风格，覆盖 UGC、开箱、奢品、病毒传播等场景
3. **端到端自动化**：从链接到成品视频，全程无需人工干预

## 三、用户流程

```
┌──────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  1. 输入 URL  │───▶│ 2. AI 提取    │───▶│ 3. 确认素材   │───▶│ 4. 选择模板   │───▶│ 5. 生成视频   │
│  或选择样本    │    │  产品数据     │    │  编辑调整     │    │  8 种风格     │    │  实时进度     │
└──────────────┘    └──────────────┘    └──────────────┘    └──────────────┘    └──────────────┘
```

### Step 1 — 输入商品链接

用户粘贴 Amazon / Shopify / 任意电商 URL，或从预设样本产品中快速选择体验。

### Step 2 — AI 自动提取产品数据

系统后台执行多阶段智能提取：

- **HTML 解析**：针对 Amazon、Shopify 等主流平台做定制化解析（JSON-LD、OG 标签、平台特征标签）
- **图片处理**：自动提取最多 10 张产品图，统一裁剪为 720×1280 竖屏画幅
- **Kimi API（Moonshot）**：智能提取简短产品名（≤20 字符）和品牌主色调（3-4 个 HEX 色值）
- **兜底机制**：API 不可用时回退到正则清洗 + 默认配色方案

### Step 3 — 素材确认与编辑

用户在 Kit Editor 中确认和微调：

- 单选产品主图
- 编辑产品名称、描述文案
- 调整品牌色
- 支持本地上传补充图片

### Step 4 — 选择广告风格模板

提供 8 套专业广告模板：

| 模板 | 风格定位 | 适用场景 |
|------|----------|----------|
| Simple UGC | 手持、真实感 | TikTok 原生广告 |
| Clean Minimal | 棚拍、360° 旋转 | 品牌官方素材 |
| Unboxing | 第一人称开箱 | 种草 / 测评 |
| Customer Reviews | 对镜口播 | 用户证言广告 |
| Viral Chaos | 高能快剪、霓虹粒子 | 病毒传播素材 |
| Luxury | 慢镜头、大理石 + 金色 | 高端品牌广告 |
| Product Story | 叙事型生活方式 | 品牌故事片 |
| Cozy Morning | ASMR 风格、暖色调 | 生活方式品类 |

### Step 5 — 视频生成与下载

- 调用视频生成 API 生成视频
- 实时进度展示
- 完成后支持在线预览 + 下载

## 四、系统架构

### 技术栈

| 层级 | 技术选型 | 选型理由 |
|------|----------|----------|
| 前端 | React + TypeScript + Tailwind | 类型安全，快速迭代 |
| 动画 | Framer Motion | 流畅的进度和过渡动效 |
| 后端 | Node.js + Express 5 | 轻量，异步友好 |
| 数据库 | PostgreSQL + Drizzle ORM | 类型安全的 ORM，schema 即代码 |
| 类型共享 | Zod schema（前后端共用） | 单一数据源，消除前后端类型不一致 |

### AI 服务集成

| 能力 | 服务商 | 模型 | 用途 |
|------|--------|------|------|
| 产品信息提取 | Moonshot (Kimi) | moonshot-v1-8k | 从原始 HTML 中提取结构化产品名 + 品牌色 |
| 视频生成 | APIYI | sora-2 | 基于 prompt + 参考图生成广告视频 |

### 关键架构决策

**异步生成 + 轮询**：视频生成请求立即返回 video ID，后台异步调用视频 API 并轮询状态。前端周期性查询进度，避免长连接超时问题。

**渐进式降级**：Kimi 不可用时回退到正则提取，视频 API Key 未配置时使用 mock 视频，确保核心流程不断裂。

## 五、当前公开版能力范围

- URL 分析与素材提取
- 产品信息编辑
- 风格模板选择
- Prompt 预览
- 视频生成接口接入
- API 缺失时的 mock 体验链路

### 当前局限

1. **单图输入限制**：视频生成参考图目前以单张为主
2. **图片裁剪影响**：固定竖屏画幅可能影响商品细节保留
3. **生成质量波动**：Prompt 模板仍需持续优化
4. **无用户体系**：当前不包含账号、历史记录、团队协作能力
